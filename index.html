Hardware Connection Specs

This dashboard is connected to a NodeMCU ESP8266 microcontroller using the following pin mapping:

Component NodeMCU Pin GPIO Pin Function
Ultrasonic Sensor 1 (Slot 1) D1 GPIO5 Trigger (TRIG)
Ultrasonic Sensor 1 (Slot 1) D2 GPIO4 Echo (ECHO)
Ultrasonic Sensor 2 (Slot 2) D3 GPIO0 Trigger (TRIG)
Ultrasonic Sensor 2 (Slot 2) D5 GPIO14 Echo (ECHO)
LED Green (Slot 1) D8 GPIO15 Vacant Status Indicator
LED Red (Slot 1) D4 GPIO2 Occupied Status Indicator
LED Green (Slot 2) D7 GPIO13 Vacant Status Indicator
LED Red (Slot 2) D6 GPIO12 Occupied Status Indicator
